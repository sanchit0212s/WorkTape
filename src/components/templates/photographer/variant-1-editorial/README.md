# variant-1-editorial

A magazine-inspired template built around the conviction that a photographer's work
reads best with the pacing of an editorial folio. Source Serif 4 (italic display)
carries the voice; Inter Tight handles small-caps metadata and navigation. The
12-column asymmetric grid cycles through three rhythm patterns so a 2-image project
and a 12-image project both feel intentional rather than filler-padded. The bio is
rendered as a drop-cap two-column paragraph — a deliberate nod to print — and each
project is framed as a numbered chapter with a bylined caption. Minimal chrome; the
photograph is always the protagonist.
