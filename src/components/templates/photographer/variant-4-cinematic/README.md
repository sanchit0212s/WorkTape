# variant-4-cinematic

A dark, full-bleed template for photographers who want the page to feel like
a film — one project per screen, sticky title card, horizontal filmstrip of
tall images per project. Fraunces italic carries the drama; Inter Tight keeps
the metadata unobtrusive. A single mix-blend-difference nav rides on top of
imagery without ever needing a colour tweak. The hero image pulls from the
first project's first frame, so minimal users still get a cinematic opening.
