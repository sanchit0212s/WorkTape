# variant-2-gallery

A white-cube, Swiss-minimal template for photographers who want the grid to
disappear. Inter Tight, hairline `#eaeaea` rules, 13-px captions with tabular
numerals, and generous 24-rem vertical rhythm. Every fourth project covers the
full 12-column width as a visual breath — enough structure to scale from one
project to twenty without ever feeling like a template. Interactive surface is
near-zero on purpose: the work answers every question.
