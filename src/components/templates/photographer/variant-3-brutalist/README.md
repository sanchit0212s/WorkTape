# variant-3-brutalist

A deliberate-friction template for photographers who want their portfolio to
feel like a catalog, not a lookbook. Archivo Black sets the display in
all-caps; JetBrains Mono handles every other piece of text. Projects live in a
high-contrast index table — each row is a keyboard-operable button that opens
the full photo set below in place. One orange accent (`#ff4b1f`) does all the
emphasis work. Designed to scale down to a single project without losing the
catalog voice.
