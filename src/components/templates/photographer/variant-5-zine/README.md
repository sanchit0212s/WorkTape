# variant-5-zine

A warm, paper-textured template for photographers whose personality IS the
product — collage pages, polaroid stamps, masking-tape accents, and a
handwritten-and-serif type pairing (Caveat + Fraunces). Cards are lightly
rotated on a forgiving 12-column grid so the layout feels made-by-hand
without collapsing at narrow widths. One project with no bio still reads as
a full zine cover; twenty projects still breathe because the size rhythm
auto-cycles. Playful, not kitsch.
